🎭 Getting started with Gooey
1 Add a device — click + in the header to enter your device's IP and port.
2 Query it — click the device tab, then go to Dashboard → Query Verbose to load its configuration.
3 Create messages — map sensor streams to OSC addresses in the Messages section.
4 Start a patch — group messages into a patch and hit Start to begin streaming.

🧭 How Orientations (Oris) Work

An ori is a saved device orientation. You can save multiple oris and the device continuously reports which one it's currently closest to.

Point ori — saved once. Matches when the device is closest to that exact orientation (geodesic distance).

Range ori — saved 2+ times. Expands into an angular sphere in quaternion space. Matches when the device is within the range radius + tolerance.

Use the Ori Only / Ori Not / Toggle conditions on messages to send different values depending on which ori is active.

Ori OnlyMessage sends only when this ori is active
Ori NotMessage sends only when this ori is NOT active
ToggleSends the high value when active, low value when not
